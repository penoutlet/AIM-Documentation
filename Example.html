Example Script

Below is an example script illustrating how to place an order using AIM. Issue a command within 7 seconds of opening AIM for Alexa to receive your command. You can extend this time by 7 seconds by issuing a command, such as adding an item to the cart. Note that you can only extend the time once beyond the initial 7 seconds. If you exceed the time limit, simply open AIM again to continue where you left off.

  1. “Alexa, open AIM."
  2. Add item number 3.
  3. Add item number 4.
  4. Show cart.”
  5. Your cart contains one item number 3 and one item number 4.”
  6. “Alexa, open AIM.
  7. Delete item number 3.
  8. Show cart.
  9. Your cart contains one item number 4.
  10. Complete order.
  11. Sending you an email to finalize your purchase.”
