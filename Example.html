Example Script



Below is an example script illustrating how to place an order using AIM. Issue a command within 7 seconds of opening AIM for Alexa to receive your command. You can extend this time by up to 7 seconds by issuing a command, such as adding an item to the cart.

  1. “Alexa, open AIM."
  2. "Add item number 3."
  3. "Add item number 4."
  4. "Show cart.”
    5. "Your cart contains one item number 3 and one item number 4.”
  5. “Alexa, open AIM."
  6. "Delete item number 3."
  7. "Show cart."
    8. Your cart contains one item number 4.
  8. "Complete order."
    9. Sending you an email to finalize your purchase.
